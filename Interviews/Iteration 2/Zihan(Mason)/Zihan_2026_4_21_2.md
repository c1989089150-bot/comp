Speaker 1: Can you tell me about how supported you feel when making course selection decisions?
Speaker 2: The university provides information like course profiles and program rules, but sometimes I still feel a bit unsure because I need to check different pages. The information is there, but it is not very easy to follow.
Speaker 1: Can you walk me through how you usually check prerequisites or course information before enrolling?
Speaker 2: I usually check the course profile first because it has most of the basic information. But sometimes I still need to open the program requirements or timetable page as well. It can be a bit annoying because I need to compare information from different places.
Speaker 1: Imagine a tool where you could select a course and immediately see its prerequisites and key details — how would you expect to use something like that?
Speaker 2: I would use it to check quickly if I am interested in one course. I can just type in the course code and immediately see whether I can take it, what prerequisites I need, and when the course is offered. That would make the decision much faster.
Speaker 1: At what point in your course selection process would a tool like this be most useful to you?
Speaker 2: I think I would use it in the early planning stage, before actually adding courses to my enrolment. I usually want to check everything first, and if the tool can help me avoid mistakes before enrolment, that would be the most helpful time.
Speaker 1: What part of a tool like this would be most valuable to you, and what other features would you want it to include?
Speaker 2: For me, the most valuable part would be showing prerequisites in a clear way. Sometimes the prerequisite rules look very simple, but when there are several conditions, it can get confusing. I would also like the tool to show timetable clashes, semester availability, and whether the course is suitable for my program.
Speaker 1: What might stop you from using a tool like this?
Speaker 2: If I still need to double check everything on the official university website, then the tool would not really save my time. I would also be less likely to use it if the design is confusing or if I cannot find information quickly.
Speaker 1: What would make you trust a website or tool to help with course selection?
Speaker 2: I would trust it more if it says the information comes from official university sources. For example, if it links back to the original course profile or program rules, I would feel more confident. It would also help if the tool shows when the information was last updated.
Speaker 1: What would make a tool like this more useful than the other course information resources currently available to you?
Speaker 2: Maybe convenience. Current sources are useful, but they are separated. If the tool can show prerequisites, semester availability, degree requirements, timetable conditions, and student feedback all on one page, I think it would be much easier to use than checking everything manually.
