Speaker 1: Can you tell me about how supported you feel when making course selection decisions?
Speaker 2: I sometimes feel a bit confused when choosing courses. There is information available, but it is not always easy to understand.
Speaker 1: Can you walk me through how you usually check prerequisites or course information before enrolling?
Speaker 2: Usually, I search the course code first. Then I open the course profile to check the prerequisites, assessments, and when the course is offered. If I do not understand something, I might ask other students.
Speaker 1: Imagine a tool where you could select a course and immediately see its prerequisites and key details — how would you expect to use something like that?
Speaker 2: I would probably use it before enrolment. I would search for the course, quickly check if I can take it, what the course is about, and whether it fits my program.
Speaker 1: At what point in your course selection process would a tool like this be most useful to you?
Speaker 2: I think it would be most useful at the beginning, when I am still planning my courses. It would help me compare different options before I make a final decision.
Speaker 1: What part of a tool like this would be most valuable to you, and what other features would you want it to include?
Speaker 2: The most valuable part would be showing the prerequisites clearly because sometimes they are hard to understand. I would also like to see timetable information, assessments, course difficulty, and some student reviews.
Speaker 1: What might stop you from using a tool like this?
Speaker 2: If the information is not updated or does not match the official university website, I probably would not trust it. Also, if the website is too confusing or slow, I would not want to use it.
Speaker 1: What would make you trust a website or tool to help with course selection?
Speaker 2: I would trust it more if it used official university information and updated regularly. It would also be helpful if it had links to the original course profiles, so I can double check the details.
Speaker 1: What would make a tool like this more useful than the other course information resources currently available to you?
Speaker 2: It would be useful if it put everything in one place. Right now, I need to check different pages, like the course profile, program rules, and timetable information. If one tool can show all of that, it would save a lot of time for me.
