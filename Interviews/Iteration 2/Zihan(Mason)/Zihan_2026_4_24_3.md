Speaker 1: Can you tell me about how supported you feel when making course selection decisions?
Speaker 2: I would say the support is okay, but it could be better. Most of the information is available online, but when I actually need to choose courses, I still feel a little bit uncertain about which course is the right one.
Speaker 1: Can you walk me through how you usually check prerequisites or course information before enrolling?
Speaker 2: I usually start by searching the course code on the university website. Then I check the course profile to see the prerequisites and assessment requirements.
Speaker 1: Imagine a tool where you could select a course and immediately see its prerequisites and key details — how would you expect to use something like that?
Speaker 2: I think I would use it when I am planning my enrolment. I would probably search a few courses and compare them. If the tool can clearly show which courses I am eligible for and which ones may cause problems, that would be really helpful.
Speaker 1: At what point in your course selection process would a tool like this be most useful to you?
Speaker 2: I think it would be most useful before enrolment, especially when I am still deciding between different courses. At that stage, I need to know which courses are available, which ones I can take, and whether they fit into my study plan.
Speaker 1: What part of a tool like this would be most valuable to you, and what other features would you want it to include?
Speaker 2: The most valuable part would be showing prerequisites in a clear way. Sometimes prerequisite rules look simple, but when there are “and” or “or” conditions, it gets confusing. I would also like the tool to show timetable clashes, assessment times, and whether the course is required for my program.
Speaker 1: What might stop you from using a tool like this?
Speaker 2: I might stop using it if the information is not accurate. For course selection, I feel like accuracy is really important because one wrong course can affect my whole study plan. Also, if the website is too messy or takes too long to load, I probably would not use it often.
Speaker 1: What would make you trust a website or tool to help with course selection?
Speaker 2: I think regular updates are important. If the course information is old, I would not feel safe using it. I would trust the tool more if it included student reviews or academic adviser comments, because that would make it feel more reliable.
Speaker 1: What would make a tool like this more useful than the other course information resources currently available to you?
Speaker 2: It would be more useful if it brought all the important information together. Right now, I might need to check the course profile, program requirements, and other pages separately. If everything is shown in one place, it would be much easier and faster to use.
